Dynamic Media System Test

This page will help you test the dynamic media management system for the Rocky Mountain Weddings website. The system allows you to update images and videos by simply replacing files in the appropriate folders, without needing to modify any code.

Note: All changes are automatically logged in src/assets/media-updates.log, providing a complete history of media updates.

Test 1: Homepage Hero Image

The homepage hero section displays a large banner image at the top of the homepage.

Steps to Test:

  1. Find a suitable image for a wedding hero banner
  2. Save it as hero-banner.jpg in src/assets/images/homepage-hero/
  3. Open the homepage (src/index.html) in your browser
  4. Verify that your new image appears in the hero section
  5. Check src/assets/media-updates.log to see if the update was recorded

Development Mode

For the best testing experience, run the website in development mode:

  1. Open a command prompt in the project root
  2. Run dev-server.bat to start the development server
  3. This will automatically refresh the page when media files are updated
